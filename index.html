How it works

The plugin takes in the words or phrases you want to type out in a data attribute, then loops through each word, splitting up the word into individual characters with a span tag which is used to add a higlight class when it is highlighted.

The plugin will replace the string of text between whatever tags you use (p tags in my case), but you don't need to start with one if you don't want to.
